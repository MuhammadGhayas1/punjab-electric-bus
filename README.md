# Punjab Electric Bus Landing Page

Marketing landing page for the Punjab Electric Bus Android app.

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- React 18

## Run Locally
```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build
```bash
npm run build
npm run start
```

## Environment
- `NEXT_PUBLIC_SITE_URL` (optional, used for canonical/SEO metadata)
